# Hack The Box (HTB) - Writeups & Solutions

A comprehensive collection of solutions and writeups for Hack The Box machines and challenges.

## 📋 Overview

This repository contains detailed walkthroughs, exploitation techniques, and solutions for various HTB machines and challenges, serving as a learning resource for cybersecurity enthusiasts and penetration testing practitioners.

## 🎯 Contents

- Machine writeups and exploitation guides
- Challenge solutions and write-ups
- Exploitation techniques and methodologies
- Security research and findings

## 🔧 Tools & Technologies

Common tools used in these writeups include:

- **Reconnaissance**: Nmap, Shodan, Recon-ng
- **Exploitation**: Metasploit, Burp Suite, Custom scripts
- **Analysis**: Wireshark, Ghidra, IDA Pro
- **Languages**: Python, Bash, PHP, etc.

## 📁 Project Structure

```
HTB/
├── machines/         # Machine writeups
├── challenges/       # Challenge solutions
├── scripts/          # Custom exploitation scripts
└── notes/            # General security notes
```

## 🚀 Getting Started

1. Clone the repository:
```bash
git clone https://github.com/fareszuleta/HTB.git
cd HTB
```

2. Browse the directories for specific machine or challenge writeups

3. Review scripts and notes for exploitation techniques
